Пояснения к решению:
* Программа компилируется с помощью скрипта build.sh
* Требования: cmake не ниже 3.25 (компилировалaсь только на gcc)
* Скомпилированная программа запускается следующим образом:
	build/programs/route <input_json> <cells_json> <output_json>
* Внешние библиотеки: библиотека работы с json в с++ от nlohmann в виде одного хэдера в папке include/nlohmann. Так как весит немного, для простоты запуска оставил в архиве с решением.
* Решение располагает ячейки на поле в виде линии. Данное расположение не является оптимальным. Однако, автор решения посчитал, что лучше неоптимальное решение, чем никакого. 
Далее прокладываются вертикальные провода от каждого контакта, который требуется соединить. Вертикальные провода соединяются горизонтальным проводом на уровне wires1. Таким образом, все провода на каждом уровне параллельны друг другу и пересечения отсутствуют.
