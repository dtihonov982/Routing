Пояснения:
* Программа компилируется с помощью скрипта build.sh
* Требования: cmake не ниже 3.25 (компилировалaсь только на gcc)
* Скомпилированная программа запускается следующим образом:
	build/programs/route <input_json> <cells_json> <output_json>
* Решение располагает ячейки на поле в виде линии. Далее прокладываются вертикальные провода от каждого контакта, который требуется соединить. Вертикальные провода соединяются горизонтальным проводом на уровне wires0. Таким образом, все провода на каждом уровне параллельны друг другу и пересечения отсутствуют.
